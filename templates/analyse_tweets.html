Or you can try with any account you want!


  • Simply enter the twitter username of the account
  • Then enter the number of tweets (max 200)
  • It will show you the six topics the account has been talking about the most, in the last 200 tweets!
  • Works best when the account is public
  • If the account isn't public, then there maybe authorization issues and the tool might not work 😭
  • For those interested, I'm using the Tweepy API to get user data.
  • I can only get the data which is public on a user's account.
